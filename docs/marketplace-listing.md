## Description

### Bring Your Diagrams to Life

Mermaid Diagrams for Confluence lets you create and render diagrams directly in your Confluence pages using the popular Mermaid.js syntax. No external tools, no image exports - just write your diagram code and see it rendered instantly.

### Supported Diagram Types

- **Flowcharts** - Process flows, decision trees, workflows
- **Sequence diagrams** - API interactions, system communication
- **Gantt charts** - Project timelines and schedules
- **Class diagrams** - Object-oriented design
- **State diagrams** - State machines and transitions
- **Entity Relationship diagrams** - Database schemas
- **Pie charts** - Data distribution
- **Git graphs** - Branch and merge visualization
- And many more Mermaid.js diagram types

### Key Features

- **Native Confluence macro** - Insert diagrams like any other Confluence element
- **4 built-in themes** - Default, Dark, Forest, and Neutral
- **External URL support** - Fetch diagrams from GitHub, GitLab, or Bitbucket with configurable refresh intervals
- **Dark mode aware** - Automatically adapts to Confluence's dark mode when using the default theme
- **Secure rendering** - Uses Mermaid.js strict security mode for safe SVG output
- **No external servers** - Runs entirely on Atlassian Forge with client-side rendering

### How It Works

1. Insert the "Mermaid Diagram" macro into your Confluence page
2. Write your Mermaid syntax in the configuration panel
3. Choose a theme (Default, Dark, Forest, or Neutral)
4. Save - your diagram renders instantly on the page

### External URL Mode

Keep diagrams in sync with your codebase. Point the macro at a raw file URL from GitHub, GitLab, or Bitbucket, and the diagram automatically updates on your configured schedule (hourly, daily, or weekly).

### Built on Atlassian Forge

This app runs entirely on the Atlassian Forge platform. Your data stays within Atlassian's infrastructure. No external servers, no data collection, no tracking.
